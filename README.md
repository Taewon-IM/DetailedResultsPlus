# DetailedResultsPlus

A mod for A Dance of Fire and Ice that enhances the results screen with detailed progress tracking, input timing analysis.

## Features

### 1. Progress Display
Enhanced the result screen to provide better context regarding your run. It now displays `Start~End%` when starting from a checkpoint, or simply `End%` for full runs.

### 2. Average Offset Analysis
Calculates and displays the average input timing offset in milliseconds directly on the results screen.

### 3. Detailed Results for Death Screen
Provides detailed results immediately upon death. This allows you to review your accuracy, progress, and timing metrics even when you fail a run.

### 4. Localization Support
Automatically switches text between English and Korean based on your in-game language settings.
Currently, only English and Korean are supported. If the game language is set to any other language, it defaults to English.

## Installation

1. Download the latest `DetailedResultsPlus_vX.X.X.zip` from the [Releases](../../releases) page.
2. Launch Unity Mod Manager and open the **Mods** tab.
3. Drag and drop the downloaded `.zip` file into the "Drop zip files here" area at the bottom.
4. Run the game and check if the mod is successfully activated in the UMM.
